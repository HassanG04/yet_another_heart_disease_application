How to use this tool

What this is

This website runs a machine learning model trained on the UCI Cleveland heart disease dataset. It estimates the probability of heart disease from clinical features you enter. It is an educational screening aid, not a diagnosis.

Using the form (Home)

  1. Open Home and enter values that match the patient or scenario you want to assess.
  2. Each field has an icon: hover or focus it for a short explanation.
  3. Categorical fields use plain language (for example Male/Female, Yes/No, or ECG types) instead of numeric codes. The correct code is sent to the model automatically.
  4. Click Analyze patient data to run the model.

Understanding the result

After analysis you will see whether the model predicts heart disease or not, together with a probability percentage labeled as Risk. The panel is color-coded:

  • Green — lower estimated risk (safer band).
  • Yellow — elevated risk; worth attention.
  • Red — higher risk; you should discuss with a clinician.
  • Dark red — high alert; prompt medical follow-up is strongly advised.

Important: Only a qualified healthcare professional can diagnose or rule out heart disease. Do not use this tool as a substitute for medical care.
