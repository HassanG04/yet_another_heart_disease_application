About the project

Business understanding

This project is a clinical assistive tool built to predict the presence of heart disease in patients based on 13 clinical features. It uses the CRISP-DM methodology to analyze the UCI Heart Disease Dataset.

The dataset

  • Source: UCI Machine Learning Repository (Cleveland dataset)
  • Features: 13 attributes including age, sex, cholesterol levels, resting ECG, etc.

The model

The backend utilizes a machine learning model (Random Forest Classifier) that has been trained on historical patient data and fine-tuned using hyperparameter optimization.

How to use this site

This is only experimental and should be used with a doctor present.
